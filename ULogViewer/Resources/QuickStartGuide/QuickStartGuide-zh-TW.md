## 選擇日誌類型
在載入並檢視日誌前，您必須先選擇適合的日誌類型。
有 3 種方式可以選擇或變更日誌類型：

### 日誌類型選單
1. 點一下工具列 ![](LogProfile_Outline_24px.png) 旁的下拉箭頭。
2. 選擇一個日誌類型。

### 日誌類型對話方塊
1. 點一下工具列的 ![](LogProfile_Outline_24px.png)。
2. 找到日誌類型。
3. 在日誌類型上點兩下，或選擇日誌類型後點一下 [確定] 按鈕。

### 以日誌類型新增分頁
1. 在分頁列上的 ![](Add_24px.png) 按一下右鍵。
2. 選擇一個日誌類型。

## 建立或編輯日誌類型
您可能需要建立或修改日誌類型以滿足您的需求。
請參考 [這裡](https://carinastudio.azurewebsites.net/ULogViewer/HowToReadAndParseLogs) 以取得更詳細關於日誌讀取的資訊。

### 編輯目前的日誌類型
1. 點一下工具列 ![](LogProfile_Outline_24px.png) 旁的下拉箭頭。
2. 點一下 ***編輯 '(名稱)' …*** 項目。請注意您不能編輯一個內建的日誌類型，您可以參考 [複製現有的日誌類型](#複製現有的日誌類型) 以複製內建日誌類型。

### 編輯日誌類型
1. 點一下工具列的 ![](LogProfile_Outline_24px.png)。
2. 找到日誌類型。
3. 將滑鼠移至日誌類型後點一下 ![](Edit_Outline_24px.png)。

### 建立新的日誌類型
1. 點一下工具列的 ![](LogProfile_Outline_24px.png)。
2. 點一下 [建立…] 按鈕。

### 複製目前的日誌類型
1. 點一下工具列 ![](LogProfile_Outline_24px.png) 旁的下拉箭頭。
2. 點一下 ***複製 '(名稱)' …*** 項目。

### 複製現有的日誌類型
1. 點一下工具列的 ![](LogProfile_Outline_24px.png)。
2. 找到日誌類型。
3. 將滑鼠移至日誌類型後點一下 ![](Copy_Outline_24px.png)。

### 匯入現有的日誌類型
1. 點一下工具列的 ![](LogProfile_Outline_24px.png)。
2. 點一下 [匯入] 按鈕。

## 開始讀取日誌
每個日誌類型皆定義了讀取日誌所需要的參數，例如一個檔案、一個目錄或 IP 端點。
若日誌類型已經設定好適當的參數 (例如執行的指令)，則您不需要額外設定參數。

以下是開始讀取日誌前您可能需要設定的參數：

### 檔案
#### 新增一個或多個檔案
有 2 種方式將檔案加入至 ULogViewer：
+ 點一下工具列的 ![](AddFile_Outline_24px.png) 並選取一個或多個檔案。
+ 從檔案總管 (macOS 上的 Finder) 拖曳一個或多個檔案至 ULogViewer 工作區中。

#### 清除所有加入的檔案
1. 點一下工具列的 ![](Delete_Outline_24px.png)。

#### 移除加入的檔案
1. 點一下側邊列的 ![](File_Text_Outline_24px.png) 開啟 ***已加入的日誌檔案*** 面板。
2. 在欲移除之檔案上點一下滑鼠右鍵。
3. 點一下 ***移除日誌檔案*** 項目。

### 工作目錄
1. 點一下工具列的 ![](Folder_Outline_24px.png)。
2. 選擇一個目錄。

### 指令
1. 點一下工具列的 ![](Terminal_Outline_24px.png)。
2. 設定欲執行之指令。

### IP 端點
1. 點一下工具列的 ![](IPAddress_Outline_24px.png)。
2. 設定 IP 位址及連接埠。

### 網路位址 (URI)
1. 點一下工具列的 ![](Uri_Outline_24px.png)。
2. 設定網路位址。

### 處理程序識別碼 (PID)
1. 點一下工具列的 ![](Process_Outline_24px.png)。
2. 設定處理程序識別碼。

### 處理程序名稱
1. 點一下工具列的 ![](Process_Outline_24px.png)。
2. 設定處理程序名稱。

## 標記日誌
您可以標記一個或多個感興趣的日誌以便稍後可以快速的找到它們。
所有標記的日誌都會在 ***已標記日誌*** 面板中列出。
若日誌是從檔案讀取，則已標記日誌的狀態將被儲存。

### 開啟已標記日誌面板
1. 點一下側邊列的 ![](Marks_Outline_24px.png)。

### 標記/解除標記日誌
有 4 種方式可以標記/解除標記日誌：
+ 選取一個或多個日誌後按下 **M** 以標記/解除標記。
+ 在選取的日誌上點一下右鍵，然後點一下 ***標記日誌>無色彩*** 或 ***解除標記日誌*** 項目。
+ 將滑鼠移動至日誌左側，然後點一下 ![](Circle_Outline_24px.png)。
+ 將滑鼠移動至日誌左側，在 ![](Circle_Outline_24px.png) 上點一下右鍵後點一下 ***無色彩*** 或 ***解除標記日誌*** 項目。

### 以色彩標記日誌
有 3 種方式可以以色彩標記日誌：
+ 選取一個或多個日誌後按下 **Ctrl+Alt+1** ~ **Ctrl+Alt+8** (macOS 為 **⌥⌘1** ~ **⌥⌘8**) 以使用您想要的顏色來標記。
+ 在選取的日誌上按一下右鍵，點一下 ***標記日誌*** 後選擇標記的顏色。
+ 將滑鼠移動至日誌左側，在 ![](Circle_Outline_24px.png) 上點一下右鍵後選擇標記的顏色。

## 篩選日誌
日誌篩選是 ULogViewer 中最重要的功能之一，它可以協助您從日誌中找出並分析問題。

### 文字篩選
日誌可以根據其顯示的日誌屬性與文字篩選來被篩選。
一個或多個文字篩選可以同時套用，而各文字篩選將以 **聯集** 方式結合。

#### 設定文字篩選
1. 按下 **Ctrl+F** (macOS 為 **⌘F**) 或點一下工具列上的文字篩選輸入欄位。
2. 以正規表示式設定文字篩選。關於您在 ULogViewer 中可以使用的正規表示式詳細資訊請參考 [這裡](https://docs.microsoft.com/dotnet/standard/base-types/regular-expression-language-quick-reference)。

當輸入焦點位於文字篩選輸入欄位時，您可以按下 **上/下** 來瀏覽該分頁中的文字篩選歷史紀錄。

#### 儲存文字篩選
有 3 種方式可以儲存文字篩選：
+ 將輸入焦點移至文字篩選輸入欄位並按下 **Ctrl+S** (macOS 為 **⌘S**)。請注意您必須先設定文字篩選。
+ 點一下工具列的 ![](Filters_Outline_24px.png) 後點一下 [建立…] 按鈕。
+ 按下 **Ctrl+P** (macOS 為 **⌘P**) 後點一下 [建立…] 按鈕。

#### 套用儲存的文字篩選
有 2 種方式可以套用儲存的文字篩選：
+ 點一下工具列的 ![](Filters_Outline_24px.png) 後選擇一個或多個文字篩選。
+ 按下 **Ctrl+P** (macOS 為 **⌘P**) 後選擇一個或多個文字篩選。

當選取文字篩選時，您可以按住 **Shift** 或 **Ctrl** (macOS 為 **⇧** 或 **⌘**) 以選取多個文字篩選。

#### 使用日誌屬性作為文字篩選
1. 在某個日誌屬性上點一下右鍵。
2. 點一下 ***以 '(值)' 篩選*** 項目然後選擇精確度。

### 等級篩選
1. 在工具列上的等級下拉選單中選取您想要看得日誌等級。

日誌類型必須定義 **等級 (Level)** 日誌屬性才可以使用等級篩選。

### 處理程序識別碼 (PID) 篩選
有 2 種方式可以設定 PID 篩選：
+ 點一下工具列上的 PID 篩選輸入欄位。
+ 在選取的日誌上點一下右鍵，再點一下 ***使用選取的 PID 篩選*** 或 ***只使用選取的 PID 篩選*** 項目。

日誌類型必須定義 **處理程序識別碼 (ProcessId)** 日誌屬性才可以 PID 篩選。

### 執行緒識別碼 (TID) 篩選
有 2 種方式可以設定 TID 篩選：
+ 點一下工具列上的 TID 篩選輸入欄位。
+ 在選取的日誌上點一下右鍵，再點一下 ***使用選取的 TID 篩選*** 或 ***只使用選取的 TID 篩選*** 項目。

日誌類型必須定義 **執行緒識別碼 (ThreadId)** 日誌屬性才可以 TID 篩選。

### 結合文字篩選與其他篩選
等級、PID 及 TID 篩選以 **交集** 結合，文字篩選以 **聯集** 結合。
而您可以透過點一下位於工具列上文字篩選輸入欄位與其他篩選欄位間的按鈕以指定文字篩選與其他篩選間的結合方式：
+ ![](FilterCombinationMode_Auto_24px.png) 自動選擇。
+ ![](Union_24px.png) 聯集。
+ ![](Intersection_24px.png) 交集。

### 暫時只顯示已標記日誌
當一個以上的篩選套用時，您可以暫時只顯示已標記日誌。
有 2 種方式可以切換顯示：
+ 按下 **Alt+M** (macOS 為 **⌥M**)。
+ 點一下工具列上的 ![](MarkedOnly_Outline_24px.png)。

### 暫時顯示所有日誌
當一個以上的篩選套用時，您可以暫時顯示所有的日誌。
有 2 種方式可以切換顯示：
+ 按下 **Alt+A** (macOS 為 **⌥A**)。
+ 點一下工具列上的 ![](Visibility_Outline_24px.png)。

### 清除所有篩選
1. 點一下工具列上的 ![](ClearFilters_Outline_24px.png)。
